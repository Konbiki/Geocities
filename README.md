# Geocities

下記のGeocitiesとNetowl(スターサーバーフリー)で公開していた内容を再公開しています。
  `http://www.geocities.co.jp/SiliconValley/7406`  
  `http://konbiki.starfree.jp`  

なお20年ぐらい前の技術情報なので、現在ではあまり使えないかもしれません。
GitHub勉強のためのリポジトリですので、内容についてはスルー願います。

---

主にプログラミングに関するTips集です。  
自分の覚え書きのようなものですが、誰かの参考になれば幸いです。  

[DirectShow関係1](https://konbiki.github.io/Geocities/Files/dshowmenu.html)  
VB6及びVB2005からDirectShowを使う方法、  
VC++6.0でオリジナルのフィルタを作る方法、  
.Net Framework用クラスライブラリ(2007/11/21更新)、など  

[VBでマウスジェスチャ](https://konbiki.github.io/Geocities/Files/mouse/index.html)  
VBでマウスジェスチャを行うためにマウスの動きを捉える方法  
DirectInputを使う方法を追加しました。  
VB2005から使う方法を追加しました。(2007/11/21)  

[OpenCV関係](https://konbiki.github.io/Geocities/Files/opencvmenu.html)  
VB(VB2005/VB2008)からOpenCVを使う方法、及び、  
OpenCVを使ったサンプルプログラムや  
OpenCVを使うときに便利なクラスライブラリなど(2008/04/08)  
新しいタイプのクラスライブラリを公開しました。(2008/12/03)  
OpenCV2.1に対応しました。(2010/04/21)  

[OpenGL関係(VB2005)](https://konbiki.github.io/Geocities/Files/opengl/index.html)  
VB(VB2005)からOpenGLを使う方法、及び、  
OpenGLを使ったサンプルプログラムなど(2006/10/30)  

[Box2D関係](https://konbiki.github.io/Geocities/Files/box2d/index.html)  
VB(VB2008)からBox2Dを使うための.NETコンポーネント(2008/06/17)  

[ARToolKit関係](https://konbiki.github.io/Geocities/Files/artoolkit/index.html)  
VB(VB2008)からARToolKitを使うための.NETコンポーネント(2008/12/25)  

[その他](https://konbiki.github.io/Geocities/Files/etc/index.html)  
GraphEditで日本語を表示させるためのツール v1.01(2012/08/22)
